### Key Features
- **Interactive UI**: Easily input top and bottom text for your memes.
- **Random Meme Templates**: Fetch random meme templates from the Imgflip API.
- **Customizable Text**: Change the text of your memes with just a few clicks.
- **Responsive Design**: Works seamlessly on various screen sizes.

- Node.js (v14 or later)

### Quick Start
```bash

git clone https://github.com/nzrrs/MemeHub.git

# Navigate to the project directory
cd memehub

# Install dependencies
```
